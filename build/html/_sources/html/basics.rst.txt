Основы HTML
===========

.. Структура документа
.. -------------------

.. .. code-block:: html  # Блок кода с подсветкой синтаксиса HTML
..     :linenos:        # Показывать номера строк (опционально)

..     <!DOCTYPE html>
..     <html lang="ru">
..     <head>
..         <meta charset="UTF-8">
..         <title>Моя страница</title>
..     </head>
..     <body>
..         <h1>Заголовок</h1>
..         <p>Абзац текста</p>
..     </body>
..     </html>

.. Основные теги
.. -------------

.. +-------------+-----------------------------------------+
.. | Тег         | Описание                                |
.. +=============+=========================================+  
.. | ``<h1>-<h6>``| Заголовки разных уровней               |  
.. +-------------+-----------------------------------------+
.. | ``<p>``      | Абзац текста                           |
.. +-------------+-----------------------------------------+
.. | ... (остальные теги) ...
